Overview

A collection of highly customizable charts and components for the web. Programmed on top of d3.js and can be used directly or within your own more advanced data visualizations.
warning

Under Development

In the early stages of development. Major design and API changes will be common. NO NOT USE IN PRODUCTION.
settings_applications
Highly Customizable
All of the charts can be customized easily to match any of your needs, whatever they may be, from color and animation to custom user interactions.
dashboard
Flexible Data Structures
By allowing the user to describe how to extract data from the supplied data structure, the charts can handle basically any data you throw at them. There is no standard data structure you need to force your data into, use your data AS IS, without modification.
extension
Easy to Expand
It's very easy to build new mChart visualizations. You can implement any pre-made chart and/or component individually or combined to make more complex visualizations.
code
ECMAScript 6
Utilizes Babel to compile next generations ECMAScript 6 source code to ECMAScript5 code, and is safe to use in all modern browsers.
color_lens
SASS
Utilizes SASS to write clearer more concise CSS, that has rich features, and keeps browser compatibility in mind.
